# Images Folder

Drop your `.jpg`, `.jpeg`, `.png`, or `.webp` images here.

The classifier (`05_image_classification/main.py`) will automatically
pick up every image in this folder and classify it.

## Example
```
images/
├── cat.jpg
├── dog.png
└── car.jpeg
```

Run:
```bash
cd hf-projects
python 05_image_classification/main.py
```
